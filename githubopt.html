github 433 可能开启了代理 
        
         If you use the vpn,
            you can try "git config --global http.proxy http://127.0.0.1:7890(7890 is the port of your vpn) "
            and "git config --global https.proxy https://127.0.0.1:7890".

            If not you can try "git config --global unset http.proxy"
            and "git config --global unset https.proxy"
